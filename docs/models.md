# Model Registry

No official shogi-variant NNUE model is published from this repository yet.
The first useful Annan model is a launch goal for the community.

When sharing a model, include:

- variant and Shogitter rule ID
- compatible repository commit
- NNUE architecture
- training config
- dataset size
- hardware
- training duration
- validation result
- `haitaka_learn verify` output
- license and download location

Do not commit large model files directly to the repository.
